     439             :     /// \brief A Boolean metafunction that evaluates whether a given

     458             :     ///     for each \c x in <tt>[0,n)</tt> and if \c Ux matches \c V

     461             :     ///     matches some \c Bx for \c x in <tt>[0,n)</tt>.
